# 快速开始

GitHub Freshness 提供 Chrome 扩展版和油猴脚本版，可以任选一种方式安装。

## Chrome 扩展版

Chrome 扩展版不需要 Tampermonkey。下载项目后，在 `chrome://extensions/` 中开启开发者模式，并加载项目的 `extension` 文件夹。

查看完整的 [Chrome 扩展安装教程](./chrome-extension.md)。

## 油猴脚本版

点击安装油猴扩展插件：https://www.tampermonkey.net/index.php

点击链接安装[GitHub Freshness](https://greasyfork.org/zh-CN/scripts/524465-github-freshness)油猴脚本：https://greasyfork.org/zh-CN/scripts/524465-github-freshness

点击右上角扩展插件中的 GitHub Freshness 设置面板，然后设置喜欢的颜色和时间阈值即可。

默认界面语言为中文。如果你使用 GitHub 英文界面，可以在设置面板中的 **当前语言** 选择 **自动** 或 **英文**；扩展和脚本都兼容中文及英文 GitHub 页面。
