# 快速开始

GitHub Freshness 提供 Chrome 扩展版和 Tampermonkey 油猴脚本版，两种版本的核心功能与设置保持一致。

## Chrome 扩展版

Chrome 扩展已经提交 Chrome Web Store，目前正在等待审核。审核通过后，本页面会补充官方安装链接。

扩展源码不包含在公开仓库中，请勿从非官方来源安装扩展文件。更多说明请查看 [Chrome 扩展版](./chrome-extension.md)。

## Tampermonkey 油猴脚本

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)。
2. 打开 [GitHub Freshness - Greasy Fork](https://greasyfork.org/zh-CN/scripts/524465-github-freshness)。
3. 点击安装或更新脚本。
4. 打开 GitHub 仓库页、目录树页面或仓库搜索页。

## 打开设置面板

可以通过以下入口打开设置：

- Chrome 扩展安装后，点击浏览器工具栏中的 GitHub Freshness 图标。
- 在仓库主页点击 Code 按钮后的 GitHub Freshness 图标。
- 在 GitHub 搜索页点击排序操作区中的 GitHub Freshness 图标。
- 油猴脚本还可以通过脚本管理器菜单中的“设置面板”打开。

![GitHub Freshness 设置面板](./img/screenshots/settings-panel.png)

## 首次配置

1. 在“主题设置”中选择要编辑的浅色或深色配置。
2. 设置新鲜期限，例如 `30 日`。
3. 调整新鲜颜色和过期颜色，按需开启背景、文字及文件图标颜色。
4. 选择当前主题和当前语言。
5. 点击“保存设置”，页面会自动刷新并应用配置。

默认界面语言为中文。使用 GitHub 英文界面时，可以把“当前语言”设为“自动”或“英文”。详细说明请查看[功能设置](./diy-settings/index.md)。
