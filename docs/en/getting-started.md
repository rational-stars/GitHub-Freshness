# Getting started

Install Tampermonkey:

https://www.tampermonkey.net/index.php

Install GitHub Freshness:

https://greasyfork.org/zh-CN/scripts/524465-github-freshness

Open the GitHub Freshness settings panel from the browser extension menu, then configure your preferred colors, time threshold, theme, and language.

If you use GitHub in English, set **Current language** to **Auto** or **English**. The script supports update-time parsing on both Chinese and English GitHub pages.
