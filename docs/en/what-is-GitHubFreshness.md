# What is GitHub Freshness?

**GitHub Freshness** is a Tampermonkey userscript that highlights GitHub repositories by update freshness.

When you browse repositories, search results, or Awesome-style lists, the script helps you quickly tell whether a project is still active. It can highlight recent and stale projects with different colors, format update dates, sort files by update time, and show repository metadata for Awesome lists.

The userscript and documentation are open source. The Chrome extension is distributed separately, and its source code is not included in the public repository.

https://github.com/rational-stars/GitHub-Freshness

## Why it exists

High-star repositories are not always actively maintained. GitHub Freshness helps you avoid spending time on stale projects by making update activity visible at a glance.

## Compatibility

- GitHub repository pages
- GitHub search pages
- GitHub repository tree pages
- Awesome-style README lists
- GitHub Chinese and English interfaces
- Common Tampermonkey-supported browsers
